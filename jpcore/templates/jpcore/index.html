これは何？

JP Core は、日本医療情報学会 FHIR 国内実装基盤研究会が開発している 日本版 HL7® FHIR® 実装ガイドです（公式: jpfhir.jp/fhir/core）。 その開発リポジトリに集まる多数の Issue を、所属 SWG ごとに効率よく改善していくための 手順書（資料）と支援ツール（issue_improver）をまとめた紹介ページです。

ツールは GitHub の Issue を一覧化し、内容から ケース A/B/C/D を判定して 着手順と spec-kit の処理プロンプトを生成します。1 Issue = 1 サイクルで、 Claude Code と spec-kit を使った改善作業をスムーズに始められます。

2つのコンテンツ

資料の閲覧

Claude の準備から Spec Kit / VS Code 環境構築、Issue のケース別手動改善、 一括処理ツールの使い方まで、導入〜運用の解説スライド（全{{ decks|length }}本）を ブラウザ内でそのままプレゼン表示できます。

資料一覧へ

ツールの模擬実行

issue_improver の読み取り専用部分をブラウザ上で体験できます。 リポジトリと SWG ラベルを入れると、Issue 一覧・ケース分類・着手順・ spec-kit プロンプトを生成します。サンプルデータでも試せます。
